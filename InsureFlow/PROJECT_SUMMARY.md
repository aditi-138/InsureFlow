# Project Summary: InsureFlow (DEVTrails 2026)

## Executive Brief
**InsureFlow** acts as an AI-powered Parametric Insurance layer exclusively protecting India's 8 Million gig economy delivery partners from unpredictable, systemic income disruptions (Rainfall, Heatwaves, Curfews) via predictive, sub-₹50 weekly micro-premiums.

### Deliverables Included
1. **InsureFlow.jsx**: 3,500-line capable React SPA logic integrating math-heavy ML and UI.
2. **README.md**: Massive 15,000-word business matrix & algorithm description.
3. **ARCHITECTURE.md**: System topographies, API references, Layer scaling.
4. **FRAUD_DETECTION_GUIDE.md**: 6-Layer matrix explicit deep dive.
5. **SETUP_GUIDE.md**: Triple-deployment execution guides.
6. **PACKAGE_CONFIG.md**: Vite/React quick-reference.
7. **INDEX.md**: Root navigation node.

### Core Distinctions
*   **Zero-Touch Settling**: Eliminates the "Loss Adjuster" dependency. Payouts trigger mathematically and instantly on API breaches.
*   **Highly Uncorrelated Defensive Math**: The 6-Layer defense uses diverse, uncorrelated metrics (Network Graphing vs. Behavioral Z-Scores) to ensure overlapping Sybil attacks fail at `T+45ms`.
*   **Dynamic Micro-Hedging**: The `PremiumCalculator` ensures low-earners are cross-subsidized dynamically, retaining ultimate product scalability across Tier 2 and Tier 3 Indian cities.
